# Architecture (draft)

Pipeline: capture -> ingest -> process -> insight -> action -> feedback

Components (first cut)
1) Data capture (RGB/multispectral, optional ground sensors)
2) Ingest (upload raw + register dataset metadata)
3) Processing (orthomosaic, indices, outputs)
4) Insights (rules/models + recommendations)
5) Delivery (report pack, optional web/dashboard)
6) Feedback (action taken + outcome)
