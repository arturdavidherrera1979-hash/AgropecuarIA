# Roadmap (draft)

Phase 0 - Discovery (now)
Goals:
- Validate pain + willingness to pay.
- Pick 1-2 crops/regions for first pilot.
- Select MVP sensor/drone stack.

Deliverables (linked to Issues):
- BM-001 Business model draft
- TECH-001 Sensor/platform research
- COMM-001 Commercial plan + pricing

Phase 1 - MVP Pilot
Goals:
- Run 1-3 pilot operations end-to-end.
- Produce insights that drive an action.
- Capture feedback loop.

Exit criteria (draft):
- Repeatable SOP for capture + processing.
- At least one decision/action driven by outputs.
- Pricing proposal accepted for next cycle.

Open questions
- Which crop/region is best for first measurable impact?
- What is the minimum data needed to deliver value?
