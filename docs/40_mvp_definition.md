# MVP Definition

Goal
- Deliver one actionable insight per plot per capture, with a repeatable SOP.

In scope
- Capture + ingest + processing for 1 crop/region
- Georeferenced outputs + basic recommendation
- Report delivery to the producer

Out of scope (for MVP)
- Multi-tenant SaaS platform
- Fully automated ML at scale

Acceptance criteria (draft)
- End-to-end cycle <= 48h after capture
- Output understandable by non-technical producer
- At least one pilot results in an action taken
